# Predictive Maintenance Analytics Dashboard

## Project Overview

This project presents a Power BI dashboard for predictive maintenance analytics, machine failure detection, technical parameter analysis and maintenance action prioritization.

The objective is to transform industrial machine data into actionable insights for maintenance, quality, production and operational decision-making.

## Business Context

In an industrial environment, machine failures can generate production losses, quality issues, maintenance costs and operational instability.

This dashboard helps identify:

- machine failure patterns;
- observations in alert status;
- technical parameters associated with risk;
- tool wear behavior;
- failure type distribution;
- maintenance and quality priorities.

## Dashboard Structure

### Page 1 — Industrial Overview

General overview of machine behavior, failures and key industrial indicators.

### Page 2 — Alerts and Risk Analysis

Analysis of observations in alert status, risk patterns and failure-related indicators.

### Page 3 — Strategic Decision-Making

Operational synthesis to support maintenance, quality and production decisions.

## Dataset

This project uses the AI4I 2020 Predictive Maintenance Dataset from the UCI Machine Learning Repository.

License: Creative Commons Attribution 4.0 International — CC BY 4.0.

The dataset is synthetic and contains 10,000 observations related to predictive maintenance and machine failure classification.

A cleaned and transformed version of the dataset is used in this project for educational and portfolio purposes.

## Main Insights

- Certain operating conditions are associated with higher machine failure risk.
- Tool wear is a relevant indicator for maintenance prioritization.
- Failure types should be monitored separately to support targeted actions.
- A dashboard is valuable only when it supports real operational decisions.
- Data analytics can connect maintenance, quality and production into one decision-making system.
